Web Frameworks in Haskell

  1. Spock is a lightweight Haskell web framework inspired by Ruby's Sinatra. It provides a full toolbox including everything to get a quick start into web programming using Haskell.
  2. Yesod Web Framework is a Haskell web framework for productive development of type-safe, RESTful, high performance web applications.
  3. Snap Framework is a simple web development framework for unix systems, written in the Haskell programming language. Snap has a high level of test coverage and is well-documented.
  4. Scotty is a Haskell web framework inspired by Ruby's Sinatra, using WAI and Warp. Scotty is the cheap and cheerful way to write RESTful, declarative web applications.
