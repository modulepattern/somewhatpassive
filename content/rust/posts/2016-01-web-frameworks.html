Web Frameworks in Rust

  1. nickel.rs is a simple and lightweight foundation for web applications written in Rust. Its API is inspired by the popular express framework for JavaScript.
  2. iron is a high level web framework built in and for Rust, built on hyper. Iron is designed to take advantage of Rust's greatest features - it's excellent type system and its principled approach to ownership in both single threaded and multi threaded contexts.
  3. hyper is a fast, modern HTTP implementation written in and for Rust. It is a low-level typesafe abstraction over raw HTTP, providing an elegant layer over "stringly-typed" HTTP.
